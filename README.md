# 🏥 Hospital Emergency Room Dashboard

## 📌 Project Overview
This project presents an interactive Power BI dashboard designed to analyze Emergency Room (ER) performance. The dashboard helps hospital management monitor patient flow, waiting times, admission rates, and overall operational efficiency.

---

## 🎯 Business Problem
Hospitals receive thousands of patients every month. Management needs an easy way to monitor:
- Patient volume
- Waiting time
- Admission rate
- Department performance
- Patient satisfaction

This dashboard provides actionable insights for better decision-making.

## 📈 Key Insights

- Emergency visits peak during weekdays.
- Average waiting time increases during busy periods.
- Most patients belong to the Adult age group.
- Admission rates vary across departments.
- Certain departments experience higher patient loads.

---

## 💡 Recommendations

- Increase staffing during peak hours.
- Reduce waiting time in high-demand departments.
- Monitor departments with low satisfaction scores.
- Improve resource allocation using monthly trends.
